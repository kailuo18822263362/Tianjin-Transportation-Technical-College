经济管理学院
魅力经院，激情无限，超越自我，勇往直前 ;
 经济管理学院始终坚持以就小业为导向，实施了“学业与职业高度融合”的人才培养模式以做实、做精、做出特色为则积极探索专业建设及课程改革
轨道与航空学院


 轨道交通学院是天津交通职业学院最年轻的分院之一，是天津交通职业学院重点发展的二级学院。
汽车工程学院


 汽车工程学院自2002年成立以来，不断深化教育教学改革，积极探索人才培养模式，实现了学业与职业的高度融合，不仅为国家和天津区域经济建设发展培养了大批汽车专业的技能型专门人才，也为学院的建设发展做出了突出贡献。
物流工程学院


 物流工程学院是中国高等职业技术教育研究会物流专业协作委员会副主任单位，被天津市人民政府授予天津滨海新区现代物流业技能型紧缺人才培养基地。
机械工程学院


 机械工程学院现设工程机械运用技术、汽车制造与装配技术、智能交通技术运用、工业机器人技术四个专业，师资力量雄厚，现有教工25人，其中天津市职业专家库入库专家2名，副教授11名，讲师6名，硕士学位以上教师8名。
路桥工程学院


 路桥工程学院筹建于2005年，主要为土木建筑类企业培养高素质技术技能人才，面向全国招生，现有在校生1000余名。
